Triply – Online Travel Booking Platform

Project Overview

I researched the travel booking market, defined the target user journey, and developed the frontend for a tour booking platform serving domestic and international travel customers.

Role & Scope

  • Business Analyst & UI/UX Developer
  • Defined customer segments and business requirements using BMC
  • Created sitemap, user flow, and interface direction in Figma
  • Developed responsive frontend pages and managed code with Git/GitHub

Key Contributions

  • Built the product structure and brand direction around a convenience-first travel experience.
  • Designed interface mockups and page layouts for the homepage, tour detail pages, cart/checkout, and admin dashboard.
  • Implemented the frontend in HTML, CSS, and JavaScript with attention to responsive design and usability.
  • Ensured the concept reflected the needs of a younger audience seeking quick and simple trip planning.

Team Members

  • Nguyễn Ngọc Như Quỳnh (Nhóm trưởng)
  • Trần Xuân Hương
  • Nguyễn Huỳnh Yến Nhi

Outcome

The project demonstrated how research, UX design, and frontend development can combine to shape a clean and usable e-commerce travel experience.
