Lost & Found Management System

A secure, transparent platform for connecting people who lose items with those who find them through smart matching and controlled handover.

NestJS Angular MongoDB JWT PWA
RoleSystem Developer & BA
Scope3-tier architecture
Core FeatureMatching engine
SecurityQR handover

Project Overview

I designed and developed a 3-tier web application that combines business analysis and fullstack implementation. The system supports item reporting, matching, moderation, and secure handover for a better lost-and-found experience.

Role & Scope

  • System Developer & Business Analyst
  • Built backend services using NestJS module-controller-service architecture
  • Developed the Angular frontend with reusable components
  • Designed business flows and UML artifacts including BPMN and Use Case diagrams

Key Contributions

  • Created a rule-based matching engine using weighted scoring for location, time, category, color, and keywords.
  • Implemented auto-moderation to filter abusive terms and reduce spam submissions.
  • Designed a QR-based handover flow and two-way token verification to ensure safe item exchange.
  • Used MongoDB to support flexible item, user, and transaction data structures.
